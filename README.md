# FreeRTOS-Cpp-Wrapper

C++-обёртка для [FreeRTOS](https://www.freertos.org/).  
Проект не является официальным и не связан с автором/компанией FreeRTOS.

> Цель — сделать работу с FreeRTOS более безопасной и удобной для C++:
> RAII, строгие типы, минимум «сырого» C-API в пользовательском коде.

---

## Возможности (задумка библиотеки)

- Обёртки над основными примитивами FreeRTOS:
  - задачи (tasks);
  - очереди (queues);
  - бинарные / счётные семафоры;
  - мьютексы (в т.ч. рекурсивные);
  - event groups и пр.
- RAII-подход:
  - создание/удаление объектов через конструкторы/деструкторы;
  - автоматическое освобождение ресурсов.
- Более безопасный интерфейс:
  - использование типов C++ вместо «голых» хэндлов;
  - упрощённая обработка ошибок (возврат `bool`, `std::optional`, и т.п.).
- Удобная интеграция с существующими проектами на FreeRTOS.

Фактический набор классов и API см. в исходниках — библиотека активно в разработке.

---

## Требования

- Наличие FreeRTOS в проекте.
- Компилятор C++ (рекомендуется C++17 и выше).
- Подходит для любых MCU/платформ, где доступны:
  - FreeRTOS;
  - C++-компилятор (например, ESP32, STM32 и т.д.).
